# 🫁 Pneumonia AI

AI-Powered Chest X-Ray Analysis using Deep Learning (VGG16 Transfer Learning)

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)
![TensorFlow](https://img.shields.io/badge/TensorFlow-2.13+-orange.svg)

## 🌟 Features

- 🏥 **Pneumonia Detection** from Chest X-Rays
- 🧠 **Deep Learning** using VGG16 Transfer Learning
- 🎯 **Confidence Score** with Visual Feedback
- 📊 **Professional Dashboard** with Streamlit
- ⚡ **Real-time Analysis** (< 2 seconds)
- 🎨 **Modern UI** with Dark Theme
